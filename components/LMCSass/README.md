# LMC-SASS

LMC-SASS is component for import .sass styles to template heading. 

```mjml
	<lmc-scss src="index.scss" inline="inline"></lmc-scss>
```

## Available props

| Prop name      | Type                                         | Default          | Required | Description                |
| -------------- | -------------------------------------------- | ---------------- | -------- | -------------------------- |
| `src`          | `string`                                     | -                | yes      | Url to sass file           |
| `inline`       | `string`                                     | -                | no       |      |
